# 🎲 バックギャモン Web アプリ

ブラウザで遊べる2人対戦バックギャモンゲームです。

## 🎮 プレイ方法

`client/index.html` をブラウザで開くだけで遊べます。インストール不要です。

1. サイコロを振るボタンをクリック
2. 動かしたいコマをクリックして選択（黄色でハイライト）
3. 移動先をクリック
4. 15枚全てベアオフしたプレイヤーの勝利！

## ✅ 実装済みルール

- コマの移動
- ヒット（相手のブロットをバーへ送る）
- バーからの再入場
- ベアオフ（超過目での上がりにも対応）
- ゾロ目（4手分）
- 合法手なし時の自動スキップ
